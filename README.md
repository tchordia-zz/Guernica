# Guernica
Project to predict artist given painting, for Kaggle competition  

## To Install:

Make sure you have git:
`sudo apt-get install git`

Clone repo:
```
git clone https://github.com/tchordia/Guernica.git ~/Guernica
cd ~/Guernica
bash install.txt
```

## To train

`cd ~/Guernica && th train.lua`
